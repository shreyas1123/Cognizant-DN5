# StudentApp

A React application for a Student Management Portal that renders class-based components: `Home`, `About`, and `Contact`. Created as part of ReactJS Hands-On Lab 2.

## Prerequisites

- Node.js (v14+ recommended)
- npm (v6+ recommended)

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

## Execution Instructions

1. Start the React development server:
   ```bash
   npm start
   ```
2. The application will be available at `http://localhost:3000`.
